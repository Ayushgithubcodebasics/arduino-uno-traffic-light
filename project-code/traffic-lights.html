Arduino Traffic Light Project

This project simulates a real traffic light using an Arduino. It includes both manual and automatic control modes. The webpage lets you test how the LEDs behave, similar to the actual circuit.

Live Traffic Light Simulation

Hardware

Components used:

  • Arduino Uno
  • Red LED (Pin 5)
  • Yellow LED (Pin 4)
  • Green LED (Pin 3)
  • 3300Ω resistors
